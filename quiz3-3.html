Let's Go, Celtics

Use internal JavaScript to implement the features below:

  • When user mouses-over any of the jerseys, the picture should change to back of that jersey (link: https://raw.githubusercontent.com/OIM3690/resources/main/templates/images/<lastname>2.webp); the heading should change to "<Full Name>'s Jersey" (without quotation marks).
  • When user mouses-out any of the jerseys, the picture should change to front of that jersey (link: https://raw.githubusercontent.com/OIM3690/resources/main/templates/images/<lastname>2.webp); the heading should change to "Let's Go, Celtics" (without quotation marks).
  • Owners of the three jerseys are, from left to right, Jayson Tatum, Jaylen Brown, Larry Bird.
  • (Optional, bonus 10 points) Create only one function for all mouseover events and only one function for all mouseout events.
